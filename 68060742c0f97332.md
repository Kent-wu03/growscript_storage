0. STRUKTUR FILE (PALING PENTING, SERING DILANGGAR)
========================
"return { ... }" WAJIB jadi SATU-SATUNYA statement di level paling luar
file, dan HARUS berada di posisi PALING AKHIR. TIDAK BOLEH ada kode
apapun (local function, local variable, dll) yang ditulis SETELAH
"return { ... }" di level file. Ini bukan soal gaya penulisan — di Lua,
return HARUS jadi statement terakhir dalam sebuah block, jadi kode yang
ditulis setelahnya akan menyebabkan SYNTAX ERROR (script gagal di-compile
sama sekali).

Kalau script asli punya banyak "local function" di level luar (log, move,
dll) yang dipakai oleh init/run/stop, function-function itu HARUS
dipindahkan ke DALAM body run(cfg) (atau init(cfg) kalau memang cuma
dipakai di situ) sebagai local function di awal function tersebut,
SEBELUM logic yang memakainya. JANGAN biarkan mereka jadi local function
di level file yang posisinya SETELAH "return { ... }".

Contoh struktur file yang BENAR (helper function pindah ke dalam run):

    info = { ... },

            Sleep(200)
        end
    end,
    stop = function(cfg)
        cfg.running = false
    end
}

JANGAN memakai "while true do ... break". HARUS memakai "while cfg.running do"
UNTUK LOOP UTAMA di run(cfg) SAJA.
Pembungkus pcall(mainLoop, cfg) di sekitar logic utama BOLEH dipertahankan
persis seperti struktur aslinya — INI POLA YANG VALID, jangan dihapus.

PENTING — JANGAN UBAH LOOP LAIN SELAIN LOOP UTAMA:
Kalau ada helper function lain (misal function pathfinding/pergerakan
seperti "move") yang punya "while true do ... end" sendiri dengan kondisi
keluar internal (misal "if x == tujuan then return true end"), loop itu
BUKAN loop utama — JANGAN diubah jadi "while cfg.running do". Biarkan
"while true do" itu apa adanya, KECUALI function tersebut memang menerima
parameter cfg. Jangan menambahkan referensi cfg.running ke dalam function
manapun yang tidak menerima cfg sebagai parameter — itu akan menyebabkan
error "attempt to index a nil value" karena cfg di function tersebut akan
merujuk ke variable global yang tidak ada.

KECUALI untuk pemanggilan MakeRequest(): kalau script asli manggil
MakeRequest() di dalam run(cfg) (langsung maupun tidak langsung lewat
fungsi lain), pemanggilan MakeRequest() itu TIDAK BOLEH ada di dalam
pcall(function() ... end) sama sekali, karena butuh coroutine context
buat nge-yield nunggu response HTTP dan bisa error "attempt to yield
across a C-call boundary" kalau dibungkus pcall. Logic lain di sekitarnya
(yang gak manggil MakeRequest) tetap boleh dibungkus pcall seperti biasa.

ids = 787 -- id seed
delay_recon = 3500
wplant = "worldkalian" -- worldplant|id kalo ada

-- do not touch
isht = true
dc = false

idb = ids-1
tstock = 0

    SendVariantList({[0] = "OnTextOverlay", [1] = "[ `^RockyHub Info `w] : " .. text})
end

function ltc(text)
    LogToConsole("`^" .. text)
end

function wrench(x, y)
    SendPacketRaw(false,{type=3,value=32,px=x,py=y,x=GetLocal().pos.x,y=GetLocal().pos.y})
end

function punch(x, y)
	SendPacketRaw(false,{type=3,value=18,px=x,py=y,x=GetLocal().pos.x,y=GetLocal().pos.y})
	Sleep(175)
end

function move(tx, ty, s, h)
    s = s or 4
    while true do
        local x = math.floor(GetLocal().pos.x / 32)
        local y = math.floor(GetLocal().pos.y / 32)
        if x == tx and y == ty then return true end
        local dx, dy, nx, ny = tx - x, ty - y, x, y
        if h then
            nx = x + math.max(-s, math.min(s, dx))
            if nx == x then ny = y + math.max(-s, math.min(s, dy)) end
        else
            ny = y + math.max(-s, math.min(s, dy))
            if ny == y then nx = x + math.max(-s, math.min(s, dx)) end
        end
        FindPath(nx, ny)
        Sleep(300 + math.random(30, 80))
    end
end

function harvesting()
    while isht do
        local world = GetWorld()
        if not world or world.name ~= wplant then
            Sleep(1300)
            ltc("Reconnecting to " .. wplant)
            RequestJoinWorld(wplant)
            Sleep(delay_recon)
            world = GetWorld()
            if world and world.name == wplant then
                dc = true
            end
        end

        if dc then
            log("Going back to pos")
            dc = false
        end

        Sleep(500)

        for _, tile in ipairs(GetTiles()) do
            if tile.fg == ids and tile.extra and tile.extra.progress == 1.0 then
                FindPath(tile.x, tile.y, 700)
                while GetTile(tile.x, tile.y) and GetTile(tile.x, tile.y).fg ~= 0 do
                    punch(tile.x, tile.y)
                end
            end
        end
    end
end

while true do
    if not isht then    
        break
    end

    local ok, err = pcall(harvesting)
    if not ok then
        ltc("Error: " .. tostring(err))
        Sleep(3000)
    end
    Sleep(200)
end
